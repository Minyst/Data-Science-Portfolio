# 원예장비 총괄생산계획 (Aggregate Production Planning)

스마트제조 강의록의 원예장비 제조업체 예제를 **Pyomo + HiGHS**로 최적화하고, **Streamlit**으로 상호작용 대시보드를 제공하는 웹앱입니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 기능

- 월별 수요, 비용(임금·재고·부재고·하청), 생산능력, 초기/최종 조건을 사이드바에서 편집
- LP(연속 완화) / IP(정수 계획) 모드 토글
- 6개 대시보드 탭
  1. **KPI**: 총비용 · 서비스수준 · 평균재고 · 평균가동률
  2. **수요·생산**: 수요 vs (생산 + 하청)
  3. **재고·부재고**: 월별 재고(+) / 부재고(−)
  4. **인력**: 인원 · 고용 · 해고 추이
  5. **비용**: 월별 스택바 + 항목별 도넛
  6. **원본표**: 전 결정변수 CSV 다운로드
- 수요 프리셋 4종 (강의록 기본 · 변동 수요 · 균등 · 성수기 집중)

## 모델 요약

결정변수 W,H,L,P,I,S,C,O (월별 인원·고용·해고·생산·재고·부재고·하청·초과시간)

비용 최소화:
```
Z = Σ 640·W + 6·O + 300·H + 500·L + 2·I + 5·S + 10·P + 30·C
```

제약:
- 노동력: `W_t = W_{t-1} + H_t − L_t`
- 생산능력: `P_t ≤ 40·W_t + O_t/4`
- 재고균형: `I_t = I_{t-1} + P_t + C_t − D_t − S_{t-1} + S_t`
- 초과근무: `O_t ≤ 10·W_t`
- 초기/최종: `W₀=80, I₀=1000, S₀=0, Iᴛ≥500, Sᴛ=0`
